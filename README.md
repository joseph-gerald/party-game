### Setting up a Workspace
![image](https://github.com/joseph-gerald/party-game/assets/73967013/587d509d-5d1b-4b43-bfb1-c755c5b32670)

$ git clone https://github.com/joseph-gerald/party-game
$ npm i
$ npm i --g nodemon
$ npm run dev
